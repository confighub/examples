# AI Start Here: Spring Platform Examples

Read [`README.md`](./README.md) first. It explains the model. This page explains how to demo it.

## Demo Pacing

1. Run one stage at a time
2. Print full output (do not summarize)
3. Explain what the output shows
4. Ask "Ready to continue?" before proceeding
