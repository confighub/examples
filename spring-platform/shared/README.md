# Shared Resources

Internal shared resources used by multiple Spring platform examples.

This directory contains files that are shared across the ADT and ADTP examples to avoid duplication while keeping each example self-contained.

## Contents

| Path | Purpose | Source |
|------|---------|--------|
| `confighub/inventory-api-*.yaml` | ConfigHub unit YAMLs for inventory-api | Generated by core example's generator |
| `field-routes.yaml` | Field ownership and mutation routing rules | From core example's operational outputs |

## Usage

Examples reference these files via `../shared/` rather than reaching into each other's directories.

## Maintenance

When the core example's generator output or field routes change, update these shared copies:

```bash
cp springboot-platform-app/confighub/inventory-api-*.yaml shared/confighub/
cp springboot-platform-app/operational/field-routes.yaml shared/
```
